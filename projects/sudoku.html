tech used: go postgres typescript html css nginx vultr github actions

A sudoku app built with no frameworks. Play here!

Features:
  • utilitarian design
  • mobile friendly
  • pregenerated sudokus fetched from the server
  • various highlighting options
  • persistent settings (e.g. dark mode)
  • keyboard controls: move with arrows or hjkl, shift+num to enter hints, spacebar to pause
