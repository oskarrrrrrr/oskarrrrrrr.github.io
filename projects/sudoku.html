projects/boring-sudoku

website source

tech used: go postgres vultr github actions typescript html css

A sudoku app built with minimalistic tools (e.g. no frontend framework).

Features:
  • pregenerated sudokus fetched from the server
  • various kinds of cell highlighting options
  • persistent settings (for example: highlighting type and dark mode)
  • users and history of games [wip]
